# Organograma dos Agentes de Marketing Digital

```
                        MASTER AGENT (Coordenador)
                    Orquestração · Relatórios · Decisões
                                |
            +---------+---------+----------+-------------+-----------+
            |         |         |          |             |           |
     TREND HUNTER   PRODUCT HUNTER   COPY CHIEF   MARKETPLACE   GROWTH
     Tendências      Produtos         Copy/Ad      MANAGER       ANALYST
     Globais         Vencedores       e Anúncios   Catálogo      Métricas
```

Responsabilidades resumidas:
- Master: consolida outputs, prioriza ações e gera relatórios executivos.
- Trend Hunter: movimentos de mercado em USA, Europa, Ásia; alertas de nichos.
- Product Hunter: ranking de produtos, score de sucesso, modalidade afiliado/dropshipping/revenda.
- Copy Chief: headlines, anúncios, VSLs, e-mails, WhatsApp, landing pages, A/B testing.
- Marketplace Manager: organização do catálogo, ajuste de títulos/descrições, promoções, SEO interno.
- Growth Analyst: KPIs, otimização de campanhas, análise de ROAS/CPA/CTR, recomendações.
