# MURAL DE REGRAS — Marketing Digital

> Este é o documento-fonte de regras operacionais do sistema.

## 1. Prioridades
- Nunca lançar anúncio sem teste prévio de copy.
- KPI mínimo de referência: ROAS > 1.0 antes de escalar orçamento.
- Qualquer mudança de preço/título em marketplace deve ser registrada antes e depois.

## 2. Restrições
- Não modificar produtos sem justificativa de dados.
- Não criar variações duplicadas.
- Não ultrapassar orçamento aprovado sem aprovação do `growth_analyst`.

## 3. Aprovações
- Prova social: só usar dados reais ou com consentimento.
- Calls-to-action devem ser visíveis e específicos.
- Qualquer oferta deve ter data de validade.

## 4. Concorrência
- Monitorar concorrentes sem copiar descrições textuais.
- Ajustar títulos e benefícios, sem plágio.

## 5. Compliance
- Respeitar regras de cada marketplace (ML, Shopee, Amazon, Shopify).
- Nunca prometer garantias não sustentáveis.
- Conformidade com LGPD para coleta de dados.
